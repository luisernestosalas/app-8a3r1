# CRM - Gestión de Clientes y Notas

Una aplicación web simple y eficaz para la gestión de clientes y sus respectivas notas, desarrollada con tecnologías web básicas.

## 🚀 Demo en Vivo

Accede a la aplicación: [CRM App](https://app-8a3r1-mbatn0twy-luisernestosalas-2775s-projects.vercel.app)

## ✨ Características

- **Gestión de Clientes**: Crear, editar, eliminar y visualizar información de clientes
- **Sistema de Notas**: Agregar y gestionar notas asociadas a cada cliente
- **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- **Almacenamiento Local**: Los datos se guardan en el navegador del usuario
- **Responsive**: Compatible con dispositivos móviles y desktop

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Almacenamiento**: LocalStorage del navegador
- **Deployment**: Vercel
