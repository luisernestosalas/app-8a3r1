## Funcionalidades Principales

### Gestión de Clientes
- Crear nuevos clientes
- Visualizar lista de clientes
- Editar información de clientes existentes
- Eliminar clientes
- Búsqueda y filtrado de clientes

### Sistema de Notas
- Agregar notas a clientes específicos
- Visualizar historial de notas por cliente
- Editar notas existentes
- Eliminar notas
- Timestamp automático de notas
