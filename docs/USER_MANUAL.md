# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Es un sistema simple para gestionar clientes y sus notas. Te permite agregar, editar y organizar información de contactos de manera fácil.

## Cómo acceder

Abre tu navegador web y ve a:

https://app-8a3r1-mbatn0twy-luisernestosalas-2775s-projects.vercel.app


## Funciones principales

### 1. Ver clientes
- Al abrir la app verás una lista de todos tus clientes
- Cada cliente muestra su información básica

### 2. Agregar un nuevo cliente
- Busca el botón **"Agregar Cliente"** o **"+"**
- Completa los campos requeridos:
  - Nombre
  - Teléfono
  - Email
  - Empresa (opcional)
- Haz clic en **"Guardar"**

### 3. Editar cliente existente
- Haz clic sobre el cliente que quieres modificar
- Cambia la información necesaria
- Guarda los cambios

### 4. Trabajar con notas
- Selecciona un cliente
- Busca la sección de **"Notas"**
- Puedes:
  - Agregar nuevas notas
  - Editar notas
